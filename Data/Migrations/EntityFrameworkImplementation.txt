Step by Step tutorial on how to implement an Entity Framework in .NET (No Server SQL):

1) Adding DbContext
    {
        public class DataContext : DbContext 
        {
            public DataContext(DbContextOptions options) : base(options)
            {

            }

            public DbSet<AppUser> Users { get; set; }

            
        }
    }

2) Creating Connection string in appsettings.Development.json and in Startup file:

    "ConnectionStrings": {
        "DefaultConnection": "Data source=datingapp.db"
    },

    AND

         public class Startup
    {
        private readonly IConfiguration _config;
        public Startup(IConfiguration config)
        {
            _config = config;
        }


        // This method gets called by the runtime. Use this method to add services to the container.
        public void ConfigureServices(IServiceCollection services)
        {
            services.AddDbContext<DataContext>(options => { <--
                options.UseSqlite(_config.GetConnectionString("DefaultConnection")); <--
            }); <--

        }    

3) Installing Entity Framework Core Tools for the .NET Command-Line Interface via a command line:
    dotnet tool install --global dotnet-ef --version 7.0.0-rc.2.22472.11


4) Create a Migration via the command line:
    dotnet migrations add InitialCreate -o Data/Migrations

5) Open NuGet Gallery and install "Microsoft.EntityFrameworkCore.Design".
